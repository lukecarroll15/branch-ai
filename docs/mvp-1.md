Colouring
On the document view, there are two critical fixes. First, if possible, please change the
background from white to cream (#F5F5DC)
If possible, it would be a great addition to let users choose between three preset background
colours — cream, white, and a soft yellow — so students can pick what works best for them.
If you get the chance, it would be worth fixing the double bullet point rendering — some list
items appear to show a hardcoded "•" character in the text content alongside the CSS list bullet,
so stripping the hardcoded character and keeping only the CSS styling would clean that up
nicely.
Highlighting
On keyword highlighting, we'd love to add a third highlight colour for sub-points like A, B, C in
lists — students asked for this explicitly in our research. Something in amber or teal sitting
alongside the existing lavender and coral would work well.
Increasing paragraph and list item spacing throughout the document view, as per students
feedback that they can find text dense.
On the quiz side, it would be great to improve the answer option styling with clear hover,
selected, and disabled states so the options feel tappable rather than static. And if possible,
increasing all tap targets on mobile to a minimum of 44x44px — particularly the quiz answer
options — would make a real difference for students using phones.
Two additional things worth considering for the pilot if there is capacity (250 euro gemini
credits). First, a "simplify this" button on any paragraph — the student taps it and Gemini
rewrites that section in plainer language on demand. This addresses a pattern we saw clearly in
user research where students were leaving the product to paste text into ChatGPT for
simplification.
Second, a simple thumbs up / thumbs down after each quiz question — easy or hard? Storing
that response is the foundation of our spaced repetition scheduling data and feeds into the
teacher dashboard later. Neither of these needs to be perfect for the pilot, just functional.
Possibly being able to adjust font size, font itself and spacing from a settings bar.
